# 📌 Project Overview

This project presents the design and implementation of a **Differential Amplifier** using an operational amplifier to produce an output voltage according to the equation:

# Vo = (Rb/Ra)(V₂ − V₁)

The resistor values were selected based on the student's roll number, where:

* **Rb** corresponds to the largest digit
* **Ra** corresponds to the smallest digit

For roll number **24L-6127**, the selected values were:

* **Rb = 7 kΩ**
* **Ra = 1 kΩ**

This resulted in an amplifier gain of:

# Gain = 7

The circuit was powered using a **±15 V supply** and designed to ensure the operational amplifier remained within its linear operating region.
